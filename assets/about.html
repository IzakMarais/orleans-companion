Orléans companion

This is a fan made companion site for the Orléans boardgame.

Orléans setup randomiser

The setup in Orléans normally requires
  1. different types of goods tiles to be shuffled together
  2. certain quantities randomly selected based on player counts
  3. certain quantities randomly removed based on player counts
  4. and the remaining tiles sorted by type again to form a supply.
The setup helper aims to bypass this shuffling and resorting by calculating randomized setup numbers for selected and removed goods tiles.
